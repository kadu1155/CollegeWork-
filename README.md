# Kadambari AI Portfolio - Wireframe V1.1

## Project Overview
This repository contains the professional HTML5/CSS3 wireframe for Kadambari's AI Portfolio. The project focuses on a clean, dark-mode inspired design tailored for a 3rd-year Computer Engineering student specializing in Backend and AI.

### Features
- **IT Girl in AI** Persona
- **Recruiter Corner**: Database-ready contact form.
- **AI Playground**: RAG demo UI placeholders.
- **Story-Mode Projects**: Grid-based project storytelling.
- **Mobile Responsive**
